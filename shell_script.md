# 起動方法
- ./script.sh
	- サブシェルが起動され、サブシェルでスクリプトが実行される
	- カレントシェルの設定の影響を受けない、与えない
	- 環境変数は引き継がれる
	- ファイルの先頭にシバン（#!/bin/bash）が必要
	- 実行権限が必要
- bash script.sh
	- シバンが不要
	- 上記以外は./script.shと同様
- . ./script.sh
	- カレントシェルでスクリプトが実行される
	- カレントシェルの設定の影響を受ける、与える
	- 実行権限が不要
- source ./script.sh
	- . ./script.shと同様
