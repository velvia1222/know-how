# キーバインド
- C：Ctrlを押しながら
- M：Meta（Altを押しながら、またはEscを押した直後）

| key | action                                      |
|:----|:--------------------------------------------|
| C-f | 1文字前に移動                               |
| C-b | 1文字後ろに移動                             |
| C-a | 行頭に移動                                  |
| C-e | 行末に移動                                  |
| M-f | 1単語前に移動                               |
| M-b | 1単語後ろに移動                             |
| C-h | 後ろの1文字削除                             |
| C-d | 前の1文字削除（文字がない場合はログアウト） |
| C-w | 1単語削除                                   |
| C-k | 行末まで削除                                |
| C-u | 行頭まで削除                                |
| C-y | 最後に削除した内容を挿入                    |
