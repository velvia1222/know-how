# Vim標準
## カーソル移動
| key      | action                                     |
|:---------|:-------------------------------------------|
| gj       | 表示上の下へ移動（折り返している時に有効） |
| gk       | 表示上の上へ移動（折り返している時に有効） |
| +        | 下の行の先頭へ移動                         |
| -        | 上の行の先頭へ移動                         |
| (        | 前の文へ移動                               |
| )        | 次の文へ移動                               |
| {        | 上の段落へ移動                             |
| }        | 下の段落へ移動                             |
| w、W     | 単語の先頭に移動（大文字は-等無視）        |
| b、B     | 単語の先頭に戻る                           |
| e、E     | 単語の末尾へ移動                           |
| ge、gE   | 単語の末尾へ戻る                           |
| :ju      | ジャンプ一覧を表示                         |
| :changes | 変更リストを表示                           |
| g;       | 前の変更位置に移動                         |
| g,       | 次の変更位置に移動                         |
| ｀.      | 最後に変更した位置へ移動                   |
| ｀^      | 最後に挿入モードから抜けた位置へ移動       |
| gi       | ｀^の位置で挿入モードに入る                |

## マーク
| key                | action                                 |
|:-------------------|:---------------------------------------|
| :marks             | マーク一覧を表示                       |
| :delmarks {a-zA-Z} | 指定したマークを削除                   |
| :delmarks!         | {A-Z0-9}を除く現バッファのマークを削除 |
| m{a-z}             | バッファローカルなマークを記録する     |
| m{A-Z}             | グローバルなマークを記録する           |
| ｀{a-zA-Z}         | 指定したマークへ移動                   |
| '{a-zA-Z}          | 指定したマークの行頭へ移動             |

## 文字検索
| key    | action                                     |
|:-------|:-------------------------------------------|
| f 文字 | 文字を検索                                 |
| F 文字 | 文字を逆方向に検索                         |
| t 文字 | 文字を検索（カーソルは文字の手前）         |
| T 文字 | 文字を逆方向に検索（カーソルは文字の手前） |
| ;      | 次の文字を検索                             |
| ,      | 次の文字を逆方向に検索                     |

## 画面表示

## 挿入モード
| key            | action                                                   |
|:---------------|:---------------------------------------------------------|
| C-x C-e        | １行下にスクロール                                       |
| C-x C-y        | １行上にスクロール                                       |
| C-o            | ノーマルモードのコマンドを一度実行して挿入モードに戻る   |
| C-r {レジスタ} | レジスタの文字列を入力                                   |
| C-r={加減乗除} | 加減乗除の研鑽結果を入力                                 |
| C-r={変数}     | 変数の値を入力                                           |
| C-u            | カーソルから行頭まで削除                                 |
| C-w            | カーソルから単語の先頭まで削除                           |
| C-t            | インデントを1段下げる                                    |
| C-d            | インデントを1段上げる                                    |
| C-y            | 上の行の同じ位置の文字を入力                             |
| C-e            | 下の行の同じ位置の文字を入力                             |
| C-a            | 前回の挿入モードの入力内容を入力                         |
| C-@            | 前回の挿入モードの入力内容を入力してノーマルモードに戻る |
| C-g j          | 挿入モードに入った位置の下に移動                         |
| C-g k          | 挿入モードに入った位置の上に移動                         |

## アドレス
| key          | action             |
|:-------------|:-------------------|
| 1            | 先頭行             |
| $            | 最終行             |
| 0            | 先頭行の前を表す行 |
| .            | 現在行             |
| 'm           | マーク行           |
| {アドレス}+n | アドレス+n行を指定 |

## レジスタ
| key    | action                                                             |
|:-------|:-------------------------------------------------------------------|
| ""     | レジスタを指定しない場合に使用される。x、d、y等で上書きされていく  |
| "0     | ヤンクの場合に上書きされる。レジスタを指定した場合は上書きされない |
| "{1-9} | 削除、変更した際に上書きされていく。1行未満の場合は上書きされない  |
| "-     | 削除、変更時に上書きされる。1行未満の場合に上書きされる            |
| "_     | 削除、変更時にレジスタに格納しない場合に指定する                   |
| "{a-z} | 指定したレジスタに格納される。上書きする                           |
| "{A-Z} | 指定したレジスタに格納される。追記する                             |
| "+     | クリップボード連携用レジスタ                                       |
| "%     | 現ファイルのファイル名                                             |
| "#     | 副ファイルのファイル名                                             |
| ".     | 最後に挿入されたテキスト                                           |
| ":     | 最後に実行されたExコマンド                                         |
| "/     | 最後に検索された文字列                                             |
| "=     | Expressionレジスタ。加減乗除が可能。変数の出力が可能               |

## マクロ
| key                      | action                                            |
|:-------------------------|:--------------------------------------------------|
| q{a-z}                   | もう一度qを押すまでに押下したキーをレジスタに登録 |
| q{A-Z}                   | レジスタに格納されたマクロに押下したキーを追加    |
| @{a-z}                   | レジスタに格納されたマクロを実行                  |
| @@                       | 最後に実行したマクロを繰り返す                    |
| {数字}q{a-z}             | 指定した回数マクロを実行                          |
| (ビジュアルモード)q{a-z} | 指定した範囲の各行についてマクロを実行            |

## ビジュアルモード
| key | action                               |
|:----|:-------------------------------------|
| V   | 行ビジュアルモードに切り替える       |
| C-v | 矩形ビジュアルモードに切り替える     |
| o   | カーソルを選択範囲の反対側に移動する |
| r   | 入力した文字で置換する               |
| p   | 選択範囲を指定したレジスタで置換する |

## コマンドラインモード
| key                           | action                                         |
|:------------------------------|:-----------------------------------------------|
| :{範囲}normal {コマンド}      | ノーマルモードのコマンドを実行                 |
| :{範囲}t{アドレス}            | コピー(レジスタを使用しない)                   |
| :{範囲}m{アドレス}            | 移動(レジスタを使用しない)                     |
| :read !{外部コマンド}         | 外部コマンドを実行し標準出力を入力             |
| {範囲}write !{外部コマンド}   | 指定した範囲を標準入力として外部コマンドを実行 |
| {範囲}!{外部コマンド}         | 外部コマンドを実行し指定した範囲を置き換える   |
| @:                            | 直前のExコマンドを繰り返す                     |
| C-r C-w                       | カーソル位置の単語を入力                       |
| C-d                           | 補完候補を表示                                 |
| q:                            | コマンドラインウィンドウを開く                 |
| q/                            | 検索用コマンドラインウィンドウを開く           |
| C-c(コマンドラインウィンドウ) | 現在行をコマンドラインに入力して戻る           |

## 文字列検索

| \h          | [A-Za-z_]                                                      |
| \H          | [^A-Za-z_]                                                     |
| \a          | [A-Za-z]                                                       |
| \A          | [^A-Za-z]                                                      |
| \l          | [a-z]                                                          |
| \L          | [^a-z]                                                         |
| \u          | [A-Z]                                                          |
