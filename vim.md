# Vim標準
## カーソル移動
| key    | action                                     |
|:-------|:-------------------------------------------|
| gj     | 表示上の下へ移動（折り返している時に有効） |
| gk     | 表示上の上へ移動（折り返している時に有効） |
| +      | 下の行の先頭へ移動                         |
| -      | 上の行の先頭へ移動                         |
| {      | 上の段落へ移動                             |
| }      | 下の段落へ移動                             |
| w、W   | 単語の先頭に移動（大文字は-等無視）        |
| b、B   | 単語の先頭に戻る                           |
| e、E   | 単語の末尾へ移動                           |
| ge、gE | 単語の末尾へ戻る                           |

## マーク
| key                | action                                       |
|:-------------------|:---------------------------------------------|
| :marks             | マーク一覧                                   |
| :delmarks {a-zA-Z} | 指定したマークを削除                         |
| :delmarks!         | {A-Z0-9}を除くカレントバッファのマークを削除 |
| m{a-z}             | バッファローカルなマークを記録する           |
| m{A-Z}             | グローバルなマークを記録する                 |
| ｀{a-zA-Z}         | 指定したマークへ移動                         |
| '{a-zA-Z}          | 指定したマークの行頭へ移動                   |
| ｀.                | 最後に変更した場所へ移動                     |

## 文字検索
| key    | action                                     |
|:-------|:-------------------------------------------|
| f 文字 | 文字を検索                                 |
| F 文字 | 文字を逆方向に検索                         |
| t 文字 | 文字を検索（カーソルは文字の手前）         |
| T 文字 | 文字を逆方向に検索（カーソルは文字の手前） |
| ;      | 次の文字を検索                             |
| ,      | 次の文字を逆方向に検索                     |

## 画面表示
| key          | action                         |
|:-------------|:-------------------------------|
| zz、z.       | カレント行を画面中央に表示する |
| zt、z<Enter> | カレント行を画面上端に表示する |
| zb、z-       | カレント行を画面下端に表示する |

## ウィンドウ
| key | action               |
|:----|:---------------------|
| C-H | ウィンドウを左に移動 |
| C-L | ウィンドウを右に移動 |

## オペレータ
| key | action             |
|:----|:-------------------|
| g~  | 大文字小文字の変換 |
| gu  | 小文字に変換       |
| gU  | 大文字に変換       |
| !   | 外部コマンドを実行 |

## テキストオプジェクト
- {オペレータ}{i or a}{オブジェクト}

### オブジェクト
| key | action               |
|:----|:---------------------|
| t   | タグに対し操作を行う |
| s   | 文に対し操作を行う   |
| p   | 段落に対し操作を行う |

## 挿入モード
| key            | action                                                   |
|:---------------|:---------------------------------------------------------|
| C-x C-e        | １行下にスクロール                                       |
| C-x C-y        | １行上にスクロール                                       |
| C-o            | ノーマルモードのコマンドを一度実行して挿入モードに戻る   |
| C-r {レジスタ} | レジスタの文字列を入力                                   |
| C-u            | カーソルから行頭まで削除                                 |
| C-w            | カーソルから単語の先頭まで削除                           |
| C-t            | インデントを1段下げる                                    |
| C-d            | インデントを1段上げる                                    |
| C-y            | 上の行の同じ位置の文字を入力                             |
| C-e            | 下の行の同じ位置の文字を入力                             |
| C-a            | 前回の挿入モードの入力内容を入力                         |
| C-@            | 前回の挿入モードの入力内容を入力してノーマルモードに戻る |
| C-g j          | 挿入モードに入った位置の下に移動                         |
| C-g k          | 挿入モードに入った位置の上に移動                         |

## アドレス
| key          | action             |
|:-------------|:-------------------|
| 1            | 先頭行             |
| $            | 最終行             |
| 0            | 先頭行の前を表す行 |
| .            | 現在行             |
| 'm           | マーク行           |
| {アドレス}+n | アドレス+n行を指定 |

## ビジュアルモード
| key | action                               |
|:----|:-------------------------------------|
| V   | 行ビジュアルモードに切り替える       |
| C-v | 矩形ビジュアルモードに切り替える     |
| o   | カーソルを選択範囲の反対側に移動する |
| r   | 入力した文字で置換する               |

## コマンドラインモード
| key                           | action                                         |
|:------------------------------|:-----------------------------------------------|
| :{範囲}normal {コマンド}      | ノーマルモードのコマンドを実行                 |
| :{範囲}t{アドレス}            | コピー(レジスタを使用しない)                   |
| :{範囲}m{アドレス}            | 移動(レジスタを使用しない)                     |
| :read !{外部コマンド}         | 外部コマンドを実行し標準出力を入力             |
| {範囲}write !{外部コマンド}   | 指定した範囲を標準入力として外部コマンドを実行 |
| {範囲}!{外部コマンド}         | 外部コマンドを実行し指定した範囲を置き換える   |
| @:                            | 直前のExコマンドを繰り返す                     |
| C-r C-w                       | カーソル位置の単語を入力                       |
| C-d                           | 補完候補を表示                                 |
| q:                            | コマンドラインウィンドウを開く                 |
| q/                            | 検索用コマンドラインウィンドウを開く           |
| C-c(コマンドラインウィンドウ) | 現在行をコマンドラインに入力して戻る           |

## 抽出
| key         | action                   |
|:------------|:-------------------------|
| :v/string/d | stringを含まない行を削除 |
| :g/string/d | stringを含む行を削除     |

## vim-surround
| key                            | action                             |
|:-------------------------------|:-----------------------------------|
| S{囲い}                        | ビジュアルモードの範囲の前後を囲う |
| ysi{オブジェクト}{囲い}        | テキストオブジェクトを囲う         |
| cs{変更前の囲い}{変更後の囲い} | 囲いを変更する                     |
| ds{囲い}                       | 囲いを削除する                     |

### 囲い
| key | action       |
|:----|:-------------|
| (   | ()空白を含む |
| )   | ()           |
| {   | {}空白を含む |
| }   | {}           |
| [   | []空白を含む |
| ]   | []           |
| ＞  | ＜＞         |
