# Vim標準
## カーソル移動
| key      | action                                     |
|:---------|:-------------------------------------------|
| w、W     | 単語の先頭に移動（大文字は記号を無視）     |
| b、B     | 単語の先頭に戻る                           |
| e、E     | 単語の末尾へ移動                           |
| ge、gE   | 単語の末尾へ戻る                           |
| +        | 下の行の先頭へ移動                         |
| -        | 上の行の先頭へ移動                         |
| (        | 前の文へ移動                               |
| )        | 次の文へ移動                               |
| {        | 上の段落へ移動                             |
| }        | 下の段落へ移動                             |
| gj       | 表示上の下へ移動（折り返している時に有効） |
| gk       | 表示上の上へ移動（折り返している時に有効） |

## 文字検索
| key    | action                                     |
|:-------|:-------------------------------------------|
| f 文字 | 文字を検索                                 |
| F 文字 | 文字を逆方向に検索                         |
| t 文字 | 文字を検索（カーソルは文字の手前）         |
| T 文字 | 文字を逆方向に検索（カーソルは文字の手前） |
| ;      | 次の文字を検索                             |
| ,      | 次の文字を逆方向に検索                     |

## 画面表示
| key          | action                   |
|:-------------|:-------------------------|
| zz、z.       | 現行を画面中央に表示する |
| zt、z<Enter> | 現行を画面上端に表示する |
| zb、z-       | 現行を画面下端に表示する |

## 挿入モード
### 挿入モードのキーバインド
| key            | action                                                   |
|:---------------|:---------------------------------------------------------|
| C-u            | カーソルから行頭まで削除                                 |
| C-w            | カーソルから単語の先頭まで削除                           |
| C-r {レジスタ} | レジスタの文字列を入力                                   |
| C-r={加減乗除} | 加減乗除の計算結果を入力                                 |
| C-r={変数}     | 変数の値を入力                                           |
| C-a            | 前回の挿入モードの入力内容を入力                         |
| C-@            | 前回の挿入モードの入力内容を入力してノーマルモードに戻る |
| C-y            | 上の行の同じ位置の文字を入力                             |
| C-e            | 下の行の同じ位置の文字を入力                             |
| C-t            | インデントを1段下げる                                    |
| C-d            | インデントを1段上げる                                    |
| C-x C-e        | １行下にスクロール                                       |
| C-x C-y        | １行上にスクロール                                       |
| C-o            | ノーマルモードのコマンドを一度実行して挿入モードに戻る   |
| C-g j          | 挿入モードに入った位置の下に移動                         |
| C-g k          | 挿入モードに入った位置の上に移動                         |

## コマンドラインモード
### コマンドラインモードのキーバインド
| key     | action                   |
|:--------|:-------------------------|
| C-r C-w | カーソル位置の単語を入力 |
| C-d     | 補完候補を表示           |

### コマンドラインモード関連
| key                          | action                                         |
|:-----------------------------|:-----------------------------------------------|
| :{範囲}normal {コマンド}     | ノーマルモードのコマンドを実行                 |
| @:                           | 直前のExコマンドを繰り返す                     |
| :read !{外部コマンド}        | 外部コマンドを実行し標準出力を入力             |
| :{範囲}write !{外部コマンド} | 指定した範囲を標準入力として外部コマンドを実行 |
| :{範囲}!{外部コマンド}       | 外部コマンドを実行し指定した範囲を置き換える   |
| :{範囲}t{アドレス}           | コピー(レジスタを使用しない)                   |
| :{範囲}m{アドレス}           | 移動(レジスタを使用しない)                     |

### コマンドラインウィンドウ
| key                           | action                               |
|:------------------------------|:-------------------------------------|
| q:                            | コマンドラインウィンドウを開く       |
| q/                            | 検索用コマンドラインウィンドウを開く |
| C-c(コマンドラインウィンドウ) | 現在行をコマンドラインに入力して戻る |

## レジスタ
### レジスタ一覧
| key    | action                                                             |
|:-------|:-------------------------------------------------------------------|
| ""     | レジスタを指定しない場合に使用される。x、d、y等で上書きされていく  |
| "0     | ヤンクの場合に上書きされる。レジスタを指定した場合は上書きされない |
| "{1-9} | 削除、変更した際に上書きされていく。1行未満の場合は上書きされない  |
| "-     | 削除、変更時に上書きされる。1行未満の場合に上書きされる            |
| "_     | 削除、変更時にレジスタに格納しない場合に指定する                   |
| "{a-z} | 指定したレジスタに格納される。上書きする                           |
| "{A-Z} | 指定したレジスタに格納される。追記する                             |
| "+     | クリップボード連携用レジスタ                                       |
| "%     | 現ファイルのファイル名                                             |
| "#     | 副ファイルのファイル名                                             |
| ".     | 最後に挿入されたテキスト                                           |
| ":     | 最後に実行されたExコマンド                                         |
| "/     | 最後に検索された文字列                                             |
| "=     | Expressionレジスタ。加減乗除が可能。変数の出力が可能               |

### レジスタ関連
| key                     | action               |
|:------------------------|:---------------------|
| :let @{レジスタ} = "〜" | レジスタの値を変更   |
| q{レジスタ}q            | レジスタの値をクリア |

## マクロ

|:-------------------------|:--------------------------------------------------|

| @{a-z}                   | レジスタに格納されたマクロを実行                  |
| @@                       | 最後に実行したマクロを繰り返す                    |
| {数字}q{a-z}             | 指定した回数マクロを実行                          |
| (ビジュアルモード)q{a-z} | 指定した範囲の各行についてマクロを実行            |

| key | action                                                         |
|:----|:---------------------------------------------------------------|

### パターン検索関連
| key           | action                                                               |
|:--------------|:---------------------------------------------------------------------|
| /〜C-r C-w    | インクリメンタルサーチで現在マッチしている単語を検索フィールドに入力 |

| /\_{特殊文字} | 改行を含む特殊文字にマッチ                                           |
| /%(〜)        | 〜をグループ化するがキャプチャしない                                 |

| //e           | マッチの末尾にカーソルを移動                                         |
| :%s///gn      | マッチ件数を表示                                                     |

## 置換
| key               | action                                   |
|:------------------|:-----------------------------------------|
| :s///c            | 一つ一つ確認                             |
